# 🤖 AI Calculator – Math Problem Solver with Gemini + LangChain + Streamlit

An interactive **AI-powered calculator** built using **Google Gemini (via LangChain)** and **Streamlit**.  
It can solve **arithmetic, algebra, probability, matrix, calculus, and reasoning-based math problems**.  
Also integrates with **Wikipedia** for general knowledge queries.

---

## 🚀 Features
- 🧮 Solve math expressions (e.g., `2+2`, `sqrt(16)`)
- 🎲 Probability & Statistics problems
- 📐 Algebra, Geometry, and Trigonometry
- 📊 Matrix operations (determinant, multiplication, etc.)
- 📖 Wikipedia search for general knowledge
- 💡 Step-by-step reasoning with Gemini
